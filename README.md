# projeto_avaliativo_python
# Sistema SEUC-4

Projeto desenvolvido em Python para monitoramento de pressão hidrodinâmica da Refinaria Delta-9.

O sistema:
- aplica ajuste térmico
- classifica zonas de estabilidade
- detecta risco crítico
- interrompe o sistema em caso de duas zonas vermelhas consecutivas
- gera métricas finais
